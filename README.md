### Documentation Libraries ###
* Socket.io - [https://socket.io/docs/]

### Build Project ###
* Install
```
git clone https://github.com/BasicMan01/snakenet snakenet
cd snakenet
npm install
```

* Build Client
```
npm run prod
```

### Start Server ###
```
node src/server/server
```
